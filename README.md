# MacOS-Shared-Profile

## Clone into existing folder (e.g. /Users/Shared)

```bash
git init
git remote add origin git@github.com:stairwaytowonderland/MacOS-Shared-Profile.git
git fetch
git pull origin main
```
## Run setup

```bash
make install
```
